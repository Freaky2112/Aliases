# Installation
### Download and change folder
```
cd ~
git clone https://github.com/Freaky2112/Aliases

cd Aliases
```
### Run install
```
./install.sh
```

## Edit Bashrc File
```
cd ~
nano .bashrc
--- change alias ssha to your ssh key (line 144)
--- you could change emoji in prompt_symbol (line 65)
```

# Uninstallation

### Change to dotfiles folder
```
cd ~/Aliases
```

### Run uninstaller
```
./uninstall.sh
```
